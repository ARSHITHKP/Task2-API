\# Task2-API



This is a simple Node.js + Express project for CRUD operations on books.



\## Features

\- GET all books

\- GET a single book by ID

\- POST (add) a new book

\- PUT (update) a book

\- DELETE a book



\## How to run

1\. Clone the repo  

2\. Run npm install  

3\. Run node server.js  

4\. Open \[http://localhost:3000/books](http://localhost:3000/books) in browser
